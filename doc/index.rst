Welcome to iTypes
=================

iTypes is a type library developed mainly for computer vision and deep learning.
It offers:

* Convenient access to various file formats and reading them directly into nuritmpy or torch tensors
* Grouping files into a dataset with a JSON specification and easily reading/wing with help the Dataset class
* TorchStruct, a data structure that is akin to dictionaries of torch tensors but can perform conversions of the data, such as dimension reordering and conversion to numpy
